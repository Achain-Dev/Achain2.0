# Achainplus
the next generation of blockchain  technique Achain 

# Get Achainplus code
git clone https://github.com/Achain-Dev/Achainplus.git --recursive

# Compile Achainplus
cd Achainplus  
./actx_build.sh

The above process takes about 20-30 minutes, depending on the network

# Install Achainplus
cd ./build  
sudo make install
